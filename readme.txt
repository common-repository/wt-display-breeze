== Description ==

A plugin that brings in your Breeze church management software data (events, full calendar, pledges, donations and contributions) for display on your WordPress website. It can be displayed via widgets on a sidebar or shortcodes within pages and posts.

This plugin is built and supported by [Worship Times](https://worshiptimes.org/ "Worship Times Websites For Ministries") and is not an official product of Breeze.

== Installation ==

1. Place the 'wt-breeze' folder in your '/wp-content/plugins/' directory.
2. Activate the Breeze Display plugin.
3. Visit 'Settings > Breeze Settings' and link to your Breeze account.
4. Change the setting to fit your needs.
5. Create or edit a page/post and use the add media > Breeze Events, Breeze Full Calendar or Breeze Campaigns link to select your data or drag the widget to your sidebar.

== Screenshots ==

1. **Options Area** -  Use this area to link to your account by adding your Breeze API Key, subdomain and the amount of future days to pull data in for.
2. **Shortcode Generator** - To drop a Breeze events list, full calendar or campaign list onto a page or a post, use the Breeze shortcode generator located in the add media area.
3. **Widget** - To drop a Breeze event or campaign list onto a sidebar, drag the widget into place and select your preferences.

== Frequently Asked Questions ==

= Do I need a Breeze account? =

Yes. This plugin pulls in from your Breeze account and uses the data you create.  To start a Breeze membership [click here](https://www.breezechms.com/ "Get started").

= Will this work with other church management software? =

No.  This plugin is developed to work with Breeze only
